## 📋 新增的功能

在左侧菜单中，你会看到**2个新增页面**：

### 1. 🔥 行业热点

**功能**：实时分析今日A股、港股、美股的热门行业

**使用步骤**：
1. 点击左侧菜单的 **"🔥 行业热点"**
2. 点击 **"🔥 分析今日热点"** 按钮
3. 等待30-60秒
4. 查看结果！

**显示内容**：
- 📊 市场情绪（牛市/熊市/中性）
- 🎯 今日关键主题
- 📈 热点行业排行榜（带热度评分）
- 💡 每个板块的关键驱动因素
- 🏆 热门股票列表
- 📥 可下载完整Markdown报告

**特色**：
- 🔴 红色标识 = 极度火热（热度≥80）
- 🟠 橙色标识 = 热门板块（热度60-79）
- 🟡 黄色标识 = 活跃板块（热度<60）
- 进度条显示热度分数

---

### 2. 🏆 行业龙头

**功能**：查找指定行业在各市场的领先企业

**使用步骤**：
1. 点击左侧菜单的 **"🏆 行业龙头"**
2. 输入行业名称（例如：科技、半导体）
3. 选择市场（A股/港股/美股，可多选）
4. 点击 **"🔍 查找龙头公司"** 按钮
5. 查看各市场的龙头公司详情

**快捷按钮**：
页面上有10个常用行业的快捷按钮，点击即可快速输入：
- 科技、金融、医疗健康、消费、能源
- 工业、房地产、材料、通信、半导体

**显示内容**：
- 💰 市值
- 📈 营收
- 📊 市场份额
- 📈 增长率
- ⭐ 竞争优势
- 📈 近期表现
- 🔗 可直接生成该公司的深度估值报告

**支持的行业**：
- 点击页面底部的 **"📚 查看所有支持的行业"** 展开查看完整列表

## 💡 使用技巧

### 技巧1：组合使用

**流程**：热点分析 → 龙头筛选 → 深度报告

1. 先在 **"🔥 行业热点"** 查看今日最热板块
2. 记下热门行业名称（如"半导体"）
3. 切换到 **"🏆 行业龙头"** 查找该行业龙头
4. 点击龙头公司的 **"生成深度估值报告"** 按钮
5. 切换到 **"📈 单公司分析"** 生成完整报告

### 技巧2：快捷操作

在 **"🏆 行业龙头"** 页面：
- 使用快捷按钮快速输入行业
- 默认已选中所有市场，可以取消不需要的
- 查看完一个行业后，可以直接输入另一个行业继续查询

### 技巧3：数据下载

- **行业热点**：点击 **"💾 生成完整热点报告"** 可下载Markdown文件
- **龙头公司**：每个公司都可以生成深度估值报告并下载

## 🔧 如果遇到问题

### 问题1：界面没有新功能

**解决**：
```bash
# 确保在正确目录
cd /Users/yilanliu/Desktop/realtimesearch-report

# 停止旧的Streamlit进程（Ctrl+C）
# 重新启动
streamlit run web_app.py
```

### 问题2：分析按钮没反应

**原因**：可能需要等待前一次分析完成

**解决**：等待当前分析完成，或刷新页面重新开始

### 问题3：显示错误信息

**查看详情**：点击 **"查看详细错误"** 展开查看具体问题

**常见原因**：
- API配置问题（检查 `config.py`）
- 网络连接问题
- 市场休市（热点数据可能为空）
